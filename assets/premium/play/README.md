# Premium Play Screen Assets

플레이 화면용 고해상도 에셋 폴더입니다.

필수 파일 계획:

- play_bg.webp: 글자 없는 플레이 화면 전체 배경, 1920x1080
- illustration_frame.webp: 왼쪽 삽화 프레임
- story_panel.webp: 중앙 스토리 패널
- witness_panel.webp: 오른쪽 증인단 패널
- top_bar.webp: 상단 정보 바
- bottom_bar.webp: 하단 목표/구절 바
- choice_button_normal.webp: 선택지 기본 버튼
- choice_button_hover.webp: 선택지 hover 버튼
- choice_button_pressed.webp: 선택지 pressed 버튼
