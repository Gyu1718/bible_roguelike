# Premium Main Screen Assets

메인 메뉴 화면용 고해상도 에셋 폴더입니다.

필수 파일 계획:

- main_bg.webp: 글자 없는 메인 메뉴 전체 배경, 1920x1080
- left_menu_panel.webp: 좌측 메뉴 패널 프레임
- chapter_panel.webp: 우측 챕터 선택 패널 프레임
- featured_chapter_frame.webp: 1장 대표 카드 프레임
- locked_chapter_card.webp: 잠긴 챕터 카드

주의:

- 배경 이미지에는 버튼 글자나 제목을 넣지 않습니다.
- 텍스트는 HTML에서 얹습니다.
- 기준 좌표는 premium-stage.css의 stage-main-* 클래스를 따릅니다.
