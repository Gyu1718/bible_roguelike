# Premium Asset Pipeline

이 폴더는 예시 화면을 웹게임에 최대한 그대로 적용하기 위한 고해상도 PNG/WebP 에셋 저장소입니다.

현재 SVG 기반 임시 에셋과 달리, 이 폴더의 목표는 다음과 같습니다.

1. 배경 일러스트는 1920x1080 WebP/PNG로 사용합니다.
2. 프레임, 패널, 버튼은 잘라낸 이미지 에셋으로 사용합니다.
3. HTML/CSS는 텍스트와 클릭 영역만 담당합니다.
4. PC 16:9 기준으로 먼저 맞추고, 모바일은 별도 축약 레이아웃으로 대응합니다.

권장 형식:

- 배경: 1920x1080 WebP
- 큰 패널/프레임: 투명 PNG 또는 WebP
- 버튼: normal / hover / pressed 3상태
- 텍스트는 이미지 안에 넣지 말고 HTML에서 얹습니다.
