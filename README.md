# 이름 없는 증인들

**성경 세계관 텍스트 로그라이크** 프로토타입입니다.

이 게임은 성경의 중심 인물을 직접 조종하지 않습니다. 플레이어는 모세, 다니엘, 라합, 예수님의 제자가 아니라, 그 사건 곁에 서 있던 이름 없는 한 사람이 됩니다.

## 1차 빌드 목표

현재 빌드는 다음을 목표로 합니다.

- 성경 사건의 중심 내용은 바꾸지 않는다.
- 유저는 주변 인물의 입장에서 선택한다.
- 선택은 스탯에 영향을 준다.
- 스탯에 따라 엔딩이 달라진다.
- 삽화는 외부 이미지 파일 없이 SVG로 통일된 성화풍/다크 판타지풍을 구현한다.
- BGM은 브라우저 Web Audio API로 간단한 저음 앰비언스를 생성한다.
- GitHub Pages에 배포할 수 있게 구성한다.

## 현재 포함된 챕터

### 1. 벽돌과 바다

핍박받던 히브리 노예가 출애굽을 경험하는 이야기입니다.

구조:

1. 벽돌 굽는 날
2. 모세라는 이름
3. 재앙의 땅
4. 문설주의 피
5. 급히 떠나는 새벽
6. 바다 앞에서
7. 벽돌 굽던 발이 바다 사이를 걷다

현재 완주 가능 챕터입니다.

### 2. 각기 옳은 대로

사사시대 무너진 마을의 생존 이야기입니다. 현재 기획 프로토타입 상태입니다.

### 3. 금령을 쓴 손

다니엘을 죽이는 법령을 베껴 쓰는 궁정 서기관 이야기입니다. 현재 기획 프로토타입 상태입니다.

### 4. 붉은 줄 아래

여리고 성 안에서 라합의 붉은 줄을 본 이웃 이야기입니다. 현재 기획 프로토타입 상태입니다.

### 5. 창을 내려놓은 자

십자가 아래에서 예수의 죽음을 목격한 로마 병사 이야기입니다. 현재 기획 프로토타입 상태입니다.

### 6. 마음이 뜨겁던 길

엠마오 길의 이름 없는 동행자 이야기입니다. 현재 기획 프로토타입 상태입니다.

### 7. 밧모의 파도

밧모섬에서 요한의 환상을 곁에서 본 감시병 이야기입니다. 현재 기획 프로토타입 상태입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 배포

GitHub Pages 배포용 워크플로가 포함되어 있습니다.

GitHub 저장소에서 다음 설정이 필요합니다.

```text
Settings → Pages → Build and deployment → Source: GitHub Actions
```

배포 주소는 보통 다음과 같습니다.

```text
https://gyu1718.github.io/bible_roguelike/
```

## 파일 구조

```text
src/main.jsx      React 진입점
src/App.jsx       게임 화면, 선택지, 저장, BGM, 삽화 시스템
src/gameData.js   챕터, 장면, 선택지, 엔딩 데이터
src/styles.css    화면 디자인
```
