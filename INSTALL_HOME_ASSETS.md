# Home Asset Pack 01 설치 안내

이 ZIP은 홈 화면 분리 에셋 연결 테스트용 패키지입니다.

## 업로드 방법

GitHub 저장소 루트에 ZIP을 푼 뒤, 나온 파일/폴더를 그대로 업로드하세요.

```text
assets/premium/home/home-bg.png
assets/premium/home/chapter-1-bg.png
assets/premium/home/chapter-2-bg.png
assets/premium/home/chapter-3-bg.png
assets/premium/home/chapter-4-bg.png
assets/premium/home/chapter-5-bg.png
assets/premium/home/ui/*.svg
home-assets.css
home-icon-asset-patch.css
```

`home-assets.css`는 기존 파일을 교체합니다.
이미지 파일을 먼저 올리지 않고 `home-assets.css`만 교체하면 이미지가 깨집니다.

## 적용 후 확인

```text
https://gyu1718.github.io/bible_roguelike/?v=84&blueprint=1
```

## 주의

이 패키지는 최종 상업 일러스트라기보다, 분리 에셋 구조가 작동하는지 확인하는 고해상도 드래프트 에셋입니다.
구조가 확인되면 같은 파일명으로 더 좋은 PNG를 교체하면 됩니다.
